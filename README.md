# Wallet Management System (C++)

A console-based application that simulates a simple digital wallet system.

The system allows users to create accounts, manage balances, perform transactions, and track financial activity in a secure environment.

## Features

### User Features
- Register and create a wallet account
- Secure login using username and password
- View current balance
- Send money to other users
- Request money from other users
- View transaction history
- Edit profile information
- Logout

### Admin Features
- Admin login
- View all registered users
- View balances and transaction history
- Add, edit, delete, or suspend user accounts
- View all transactions in the system
- Adjust user balances

## Data Persistence

User data and transactions are stored in **files**, allowing the system to preserve data even after the program is closed and reopened.

## Error Handling

The system includes basic error handling for cases such as:

- Insufficient balance
- User not found
- Invalid login credentials
- Suspended accounts

## Technologies

- C++
- File Handling
- Data Structures

## Learning Objectives

This project focuses on:

- Applying data structures in real systems
- Implementing authentication and transaction logic
- Managing persistent data using files
- Practicing problem solving and system design
